# AnalystLab Africa — Data Analytics Internship

This repository documents my work through the AnalystLab Africa Data Analytics Internship — an 8-week hands-on program covering the full analytics workflow, from raw data cleaning through SQL, dashboarding, and business case studies.

## 📅 Weeks Completed

### [Week 1-2: Data Cleaning & EDA](./Week1-2_DataCleaning_EDA)
Cleaned and analyzed two real-world datasets (500K+ e-commerce transactions and a Netflix content catalog) in Python — handling missing values, duplicates, and formatting issues, performing EDA, and building visualizations and business insights.

### [Week 3: SQL & Data Querying](./Week3_SQL)
Wrote 25 SQL queries across two databases (Chinook music store + Sample Sales Data) in MySQL — covering joins, subqueries, window functions, and query optimization to answer real business questions.

## 🛠️ Tools Used So Far
Python (Pandas, Matplotlib), Jupyter Notebook, MySQL, MySQL Workbench

## 📌 More weeks coming soon
This repository will continue to grow with each week of the internship, covering data visualization/dashboarding (Power BI/Tableau), business analytics case studies, advanced Python & time-series analysis, ETL pipelines, and a final capstone project.
