# AnalystLab Africa — Week 1-2: Data Cleaning & EDA

Data analytics internship task covering data cleaning, exploratory data analysis (EDA), and visualization on two real-world datasets.

## 📊 Datasets
- **E-Commerce Transactions** (UK online retailer, 500K+ rows) — [source](https://www.kaggle.com/datasets/vijayuv/onlineretail)
- **Netflix Movies & TV Shows** — [source](https://www.kaggle.com/datasets/shivamb/netflix-shows)

## 🧹 What was done
- Identified and handled missing values, duplicates, and inconsistent formats
- Standardized column names, data types, and date formats
- Performed data validation — caught and corrected a real data-entry error (duration values mistakenly stored in the Netflix "rating" column)
- Conducted exploratory data analysis and built visualizations (bar charts, line charts, histograms, pie charts)
- Extracted 5 key insights per dataset
